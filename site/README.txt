מתכונת · מדריך האש — חבילת אירוח
=================================
כדי שהאייקון יעבוד על מסך הבית של אנדרואיד, צריך "להעלות לאינטרנט" פעם אחת.
בלי טרמינל, בלי קוד.

הדרך הכי פשוטה (ממחשב, העלאה ידנית):
1. היכנס ל-  dash.cloudflare.com  ->  Workers & Pages  ->  Create  ->  Pages  ->  Upload assets
2. גרור את index.html + התיקייה site/ לתוך החלון
3. תקבל קישור (כתובת https://<שם>.pages.dev)
4. פתח את הקישור בטלפון ב-Chrome
5. תפריט (⋮) -> "התקן אפליקציה" -> אישור
   => אייקון האש יופיע על מסך הבית ויפתח את האפליקציה במסך מלא.

אוטומטי (מומלץ, ללא העלאה ידנית):
- חבר את מאגר ה-GitHub ל-Cloudflare Pages עם פקודת-בנייה `python build.py`.
  כל דחיפה ל-main תבנה ותפרסם מחדש אוטומטית.

חשוב: זה חייב לעבוד דרך הקישור (https), לא דרך פתיחת הקובץ ישירות.
הנתונים שלך (מועדפים, יומן, מזווה) נשמרים בכתובת הזו -> השתמש בכלי
"גיבוי ושחזור" שבאפליקציה אם תעבור כתובת.
